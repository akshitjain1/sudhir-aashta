# Sudhir & Aastha Wedding Invitation

A premium, cinematic, and mobile-first wedding invitation web application. 
This project was designed with a focus on luxury aesthetics, smooth animations, and an immersive user experience utilizing React, Tailwind CSS, and Framer Motion.

## 🌟 Key Features
- **Cinematic Reveal:** Realistic, fabric-textured animated curtains that open sequentially to reveal the main stage.
- **Micro-Interactions:** Custom hover states, glowing text, and staggered Framer Motion entrances.
- **Ambient Petals:** A custom Canvas-based depth-simulated falling petals system.
- **Audio Experience:** Integrated background music with smooth fade-in/fade-out toggles.
- **Live Countdown:** Real-time countdown timer tracking directly to the wedding date.

## 💻 Tech Stack
- **React 19** 
- **Tailwind CSS v4** (Utility-first styling with custom tokens)
- **Framer Motion** (Cinematic & micro-animations)
- **Vite** (Next-generation, ultra-fast frontend tooling)

## 🚀 Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/akshitjain1/sudhir-aashta.git
   ```

2. Navigate into the project:
   ```bash
   cd sudhir-aashta
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🌐 Deploying to Vercel

This project is 100% production-ready and optimized for Vercel deployment. No server-side changes are necessary; it compiles fully to static files.

1. Create an account on [Vercel](https://vercel.com/) and link it to your GitHub.
2. Click **Add New Project** and import the `sudhir-aashta` repository.
3. Vercel will automatically detect `Vite` for the framework optimization. Let the default configuration run:
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
4. Click **Deploy**. Within seconds, your beautiful wedding site will be live!
